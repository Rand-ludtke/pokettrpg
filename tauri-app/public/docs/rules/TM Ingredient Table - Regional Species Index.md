## How to use this index
This is the broad lookup table for **regional Sage, CAP, and Custom species** in the Continent A roster.

Use it when the party harvests from a species that is not already covered in the shorter type-lane sheet.

Rules:
- The **Default harvest part** is the fast, table-safe part profile for that species.
- Dual-type species can still be harvested toward either of their own type lanes, but the default part gives the quickest ruling.
- If a scene strongly suggests a different body part, keep the same type lanes and swap the part description.
