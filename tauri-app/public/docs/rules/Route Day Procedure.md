Use this when you want overland travel to feel consistent instead of hand-waved.

## One half-day block

Run travel in this order:

1. **Pick the active travel team**
   - decide which Pokemon are out and helping
   - if there is no active route variation, trainers can swap through the Pokedex / storage sync now

2. **Apply base travel drain**
   - each trainer loses `1d12 SP`
   - each helping Pokemon loses `1d6 SP`

3. **Roll route variation**
   - roll the route's **Seam Pressure d20** by stability: natural 1 is High Chaos, otherwise read No Variation, Mild, Standard, or Wild
   - if a variation happens, roll the **Biome Shift d6** to see whether the route temporarily changes biome
   - if the result is a variation, roll that route's matching **d6 chart**
   - if the result is High Chaos, roll that route's **d4 High Chaos chart**

4. **Run one main obstacle or route scene**
   - navigation
   - weather
   - footing
   - fracture weirdness
   - NPC trouble
   - ambush or wild encounter

5. **Apply extra strain**
   - No Variation `0 SP`
   - Mild `1d4 SP`
   - Standard `1d6 SP`
   - Wild `1d8 SP`
   - High Chaos `1d12 SP`
   - failed checks usually add `+2 SP`

6. **Resolve help from Pokemon**
   - if a Pokemon meaningfully helped, mark a **Bond Scene** for it
   - if a trap, gadget, or crafted item mattered, note the spent use now

7. **Gather or salvage**
   - allow one meaningful gather attempt from the current biome or scene
   - if the party is at usable water, one trainer can fish instead of making a normal gather attempt

8. **Check distance covered**
   - if a trainer or required mount hit `0 SP`, only half distance is covered
   - if someone passed out, travel stops or becomes a survival scene

9. **Short rest or press on**
   - if the party stops, restore half SP
   - allow 1 meaningful craft during the break

## End of day

At camp or shelter:

1. long rest
2. medicine and field treatment
3. 2 quick crafts or 1 meaningful craft plus 1 quick craft
4. resolve Bond Scene rolls for Pokemon that helped during travel
5. update supplies, damaged gear, and route notes

## Table rhythm goal

Each half-day should usually produce:

- 1 route complication
- 1 resource consequence or cost
- 0 to 1 encounter
- 0 to 1 gather result
- 0 to 1 Pokemon bond moment

That is enough to make travel matter without turning it into accounting.
